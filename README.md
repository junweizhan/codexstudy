# 测试GitHub

这是一个轻量的练习仓库，用于演示：

- 基础 Git 提交流程
- Python 脚本的简单输出
- 中文目录与文件协作

## 项目结构

```text
.
├── README.md
└── 测试test/
    └── hello.py
```

## 快速开始

### 1) 运行示例脚本

```bash
python3 测试test/hello.py
```

你将看到多行文本输出，其中包含：

- `what can i say`
- `hello python!`
- `hello java`

## 代码片段

项目中包含一条示例代码：

```python
print('what can i say')
```
